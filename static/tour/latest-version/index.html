--- title: Get the latest version ---

Once you have the link to the latest release, you can fetch its metadata using the get version endpoint using the URL from links.latest_version.

Example

Make a GET request to https://api.beta.ons.gov.uk/v1/datasets/cpih01/editions/time-series/versions/6
